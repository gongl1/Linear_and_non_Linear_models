We will train a Proportional Hazard model to identify important prognostic factors from the nine explanatory variables. We will use the Backward Selection method to exclude non-significant predictors.

We first include all nine explanatory variables in the model. After we have trained a Proportional Hazard model, we will retrieve the summary object to obtain the test significance values (i.e., the p-value) of the explanatory variables.

Next, we will look among the explanatory variables whose p-values are more than 0.15. Then, remove the explanatory variable, if any, that has the highest p-value from the model. We will repeat the Backward steps until there are no more explanatory variables that we can remove.

# Load dataset
df = pd.read_csv('myeloma.csv') 

# Define the Cox Proportional Hazards model
cph = CoxPHFitter()

# List of all explanatory variables
variables = ['LogBUN', 'HGB', 'Platelet', 'Age', 'LogWBC', 'Frac', 'LogPBM', 'Protein', 'SCalc']

# Initial fit with all variables
X = df[variables + ['Time', 'VStatus']]
cph.fit(X, duration_col='Time', event_col='VStatus')

# Initialize a list to store the step summary
steps_summary = []

# Backward selection process
while True:
    # Get the summary of the current model
    summary = cph.summary

    # Filter variables with p-value greater than 0.15
    non_significant = summary[summary['p'] > 0.15]

    if not non_significant.empty:
        # Find the variable with the highest p-value
        variable_to_remove = non_significant['p'].idxmax()
        highest_p_value = non_significant['p'].max()

        # Add step information to summary
        steps_summary.append({'Variable Removed': variable_to_remove, 'P-Value': highest_p_value})

        # Remove the variable with the highest p-value
        variables.remove(variable_to_remove)

        # Refit the model without the removed variable
        X = df[variables + ['Time', 'VStatus']]
        cph.fit(X, duration_col='Time', event_col='VStatus')
    else:
        # If there are no variables to remove, break the loop
        break

# Convert steps summary to a DataFrame for easy viewing
steps_summary_df = pd.DataFrame(steps_summary)

# Print the step summary table
print(steps_summary_df)

print("-------------------------------------------------------------")

# Print the final model's summary
print(cph.summary)
